# Fire Safety Solutions Colombia

Sitio web construido con Vite, React, TypeScript, Tailwind CSS y componentes de shadcn/ui.

## Requisitos

- Node.js
- npm

## Desarrollo local

```sh
npm install
npm run dev
```

## Scripts disponibles

- `npm run dev`: inicia el servidor de desarrollo
- `npm run build`: genera la versión de producción
- `npm run preview`: previsualiza la build
- `npm run test`: ejecuta las pruebas
